# GDP Analysis Web Application

## Overview
Converting the Python GDP analysis into a modern web application using the FastAPI + Vue3 + Three.js stack.

## Tech Stack
- **Backend:** FastAPI
- **Frontend:** Vue 3
- **3D Visualization:** Three.js
- **Additional:** Chart.js for 2D visualizations
